Bonsai Help

Overview

  • Generate and refine stylized bonsai models in real-time.
  • Use Bonsai, Scene, Render, Gallery, Export, and Settings tabs to control the app.

Navigation

  • Drag to orbit camera.
  • Mouse wheel or pinch to zoom.
  • Use Frame to re-center and fit the tree.

Main Hotkeys

  • Space play/pause rotation
  • F frame bonsai
  • X screenshot
  • R randomize all properties
  • Enter save snapshot
  • [/] hold manual spin
  • P tips overlay
  • O, T, C, S, D debug toggles
  • \ debug master toggle
  • Esc close Help modal
